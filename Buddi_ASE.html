Buddi
Supplementary Material -- ASE 2018
Description
  • Language: Java
  • Version: 3.4.0.8
  • Buddi is a personal financial tool for managing a budget. It is geared for those with little financial background. (http://buddi.digitalcave.ca). This benchmark is based on Buddi version 3.4.0.8, the lastest version as of Feb 13th, 2011. We downloaded this version of Buddi, obtained its GUI structured using the GUIRipper, generated its Event-Flow Graph (EFG).
