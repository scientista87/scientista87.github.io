TerpSpreadSheet
Supplementary Material -- ASE 2018
Description
  • Language: Java
  • Version: 3.0
  • TerpSpreadSheet is a simple Java application for processing spreadsheets, drawing graphs and charts (similar to MS Excel). The program was developed by senior Software Engineering students at the University of Maryland, College Park as term projects. This program has multiple versions developed by different teams in several semesters. This benchmark is based on TerpSpreadSheet version 3.0. We downloaded this version of TerpSpreadSheet, obtained its GUI structure, generated Event-Flow Graph (EFG).
